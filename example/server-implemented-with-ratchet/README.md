# PHP WebSocket Chat Server (Ratchet version)

This is the same chat server as the one from the 'server' folder, but this one was implemented with [Ratchet](https://github.com/ratchetphp/Ratchet).

## Try it out!

Firt you need to use composer to install the dependencies. So download composer, then run: `php composer.phar install` ([Learn more about it](https://packagist.org/))  
Then, in this folder, run the command line: `php ChatServer.php`  
Then, on your browser, access: `http://localhost/path/to/php-websocket-server/example/client/`
